## 待观察重点

- 策划 Agent 是否能像 Agent 一样主动检索、读文件、写资料、检查并收束，而不是只普通聊天。
- Agent 运行中停止是否有效，是否会出现陈旧的“运行中”状态。
- 工具过程是否默认自然折叠，展开后是否能看到关键证据。
- 模型切换、提供商错误诊断、余额/模型不存在/baseUrl 错误提示是否清楚。
- 角色扮演配置保存后是否成为本书默认配置，草案不应污染默认配置。
- 角色扮演上下文是否有角色可见性隔离、世界书触发日志和 transcript 审计。
- 行文页验收链、修复预览、低风险应用和复检是否可用。
- RAG 测试集是否能反映世界书触发、角色可见性、记忆写入和改稿偏好边界。
- 粒子层、悬浮栏、工具提示是否存在遮挡、重复提示或视觉突兀。
- 长历史线程是否按需加载和压缩，不应一次性铺满页面或把大工具输出塞进消息正文。
